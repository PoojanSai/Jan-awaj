# Voice-Based Citizen Grievance Redressal System

A full-stack web-based civic grievance platform using voice input.

## Features
- Voice complaint submission
- Speech-to-text via AssemblyAI
- Department identification
- Location-based routing
- PostgreSQL storage
- Email forwarding (test mode)

## Tech Stack
- FastAPI
- PostgreSQL
- AssemblyAI
- HTML, JS (MediaRecorder API)

## Deployment
Backend hosted on Render
